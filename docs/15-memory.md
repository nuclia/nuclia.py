# Memory

Nuclia Memory provides persistent, queryable memory for AI agents backed by Nuclia KnowledgeBoxes.

## Core Concepts

- **Memory**: A memory instance maps to a Nuclia KnowledgeBox. It stores and retrieves discrete units of information.
- **Topic**: A discrete unit of memory stored as a Nuclia resource. Each topic has a unique ID, an optional slug, a title, and one or more content fields.
- **Annotation**: A conversational note attached to a topic by a specific user. Annotations store structured content (text, reasoning, context) as a conversation field on the resource.
- **Fact**: An auto-generated, objective piece of information extracted from an annotation. Facts are stored separately from annotations and are generated by the system.
- **Recall**: A generative query that returns an AI-generated answer grounded in stored topics.

## Usage

### Setup

```python
from nuclia import sdk

memory = sdk.NucliaMemory()
```

### Storing information (`store`)

Store a new topic from text:

```python
memory.store("The deployment uses GitHub Actions.")
```

Store with a title and slug:

```python
memory.store(
    "The deployment uses GitHub Actions.",
    title="Deployment process",
    slug="deployment-process",
)
```

Store with a summary:

```python
memory.store(
    "The deployment uses GitHub Actions.",
    title="Deployment process",
    slug="deployment-process",
    summary="Overview of the CI/CD pipeline.",
)
```

Store from a local file:

```python
memory.store(path="/path/to/file.pdf", title="Architecture Diagram")
```

Store from a URL:

```python
memory.store(url="https://example.com/policy", title="Company Policy")
```

Append content to an existing topic:

```python
memory.store(
    "Rollbacks are handled via git revert.",
    topic="deployment-process",
)
```

### Querying memory (`recall`)

Ask a question and get a generative answer grounded in a specific topic:

```python
result = memory.recall("How does deployment work?", topic="deployment-process")
print(result.answer)
```

Scope the search to a specific user's annotations:

```python
result = memory.recall(
    "How does deployment work?",
    topic="deployment-process",
    who="alice@example.com",
)
```

Include all users' annotations and facts in the search context:

```python
result = memory.recall(
    "How does deployment work?",
    topic="deployment-process",
    who="*",
)
```

Pass prior conversation as context:

```python
from nucliadb_models.search import ChatContextMessage

result = memory.recall(
    "What about rollbacks?",
    topic="deployment-process",
    context=[
        ChatContextMessage(author="user", text="How does deployment work?"),
        ChatContextMessage(author="assistant", text="It uses GitHub Actions."),
    ],
)
```

Access citations from the answer:

```python
for footnote_id, citation in result.citations.items():
    print(f"[{footnote_id}] {citation.text}")
```

### Annotating topics (`annotate`)

Append a note to an existing topic:

```python
memory.annotate(
    topic="deployment-process",
    text="Confirmed with the team on 2025-01-15.",
)
```

Include context messages and reasoning:

```python
from nuclia.sdk.memory import AnnotationContextMessage

memory.annotate(
    topic="deployment-process",
    text="Needs review after next sprint.",
    context=[
        AnnotationContextMessage(author="alice", text="Should we update the pipeline?"),
    ],
    reasoning="The current pipeline lacks a staging step.",
    who="alice@example.com",
)
```

### Retrieving a topic (`get`)

Get a topic by ID or slug (basic metadata only):

```python
topic = memory.get(topic="deployment-process")
print(topic.title)
print(topic.summary)
```

Fetch the most recent annotations and/or facts:

```python
topic = memory.get(topic="deployment-process", annotations=10, facts=5)

for annotation in topic.annotations:
    print(annotation.content.text)
    print(annotation.content.reasoning)
    print(annotation.timestamp)

for fact in topic.facts:
    print(fact.text)
```

### Listing topics (`list`)

```python
page = memory.list(query="deployment", page=0, size=10)
for topic in page.items:
    print(topic.id, topic.title)
print(f"Total: {page.total}, Has more: {page.has_more}")
```

### Deleting (`forget`)

Delete a specific annotation on a topic:

```python
memory.forget(topic="deployment-process", annotation="<annotation-id>", who="alice@example.com")
```

Delete all annotations by a user on a topic:

```python
memory.forget(topic="deployment-process", annotations=True, who="alice@example.com")
```

Delete an entire topic (irreversible — requires confirmation):

```python
memory.forget(topic="deployment-process", confirm=True)
```

## Data Models

| Class | Description |
|-------|-------------|
| `Topic` | A discrete unit of memory (`id`, `slug`, `title`, `summary`, `annotations`, `facts`). |
| `Annotation` | A single annotation (`id`, `timestamp`, `content`, `attachments`). |
| `AnnotationContent` | Structured annotation content (`text`, `reasoning`, `context`). |
| `AnnotationContextMessage` | A context message within an annotation (`author`, `text`). |
| `Fact` | An auto-generated fact extracted from an annotation (`id`, `timestamp`, `text`). |
| `RecallResult` | Result of a generative recall (`answer`, `citations`). |
| `RecallCitation` | A single citation from a recall result (`id`, `text`). |
| `TopicPage` | Paginated listing of topics (`items`, `total`, `has_more`). |

## Async

An async version is available via `AsyncNucliaMemory`. The method signatures mirror the sync API:

```python
from nuclia import sdk

memory = sdk.AsyncNucliaMemory()

await memory.store("The deployment uses GitHub Actions.", slug="deployment-process")
result = await memory.recall("How does deployment work?", topic="deployment-process")
print(result.answer)
```
